API Endpoints

GET Health Check

Check if the API is running and healthy.

Endpoint:
/api/health
Response Example:
{
  "status": "healthy",
  "message": "API is running successfully",
  "timestamp": "2024-01-01T00:00:00Z"
}

GET Hacker News

Fetch front page Hacker News stories.

Endpoint:
/api/hacker-news
Response Example:
{\n  \"status\": \"success\",\n  \"count\": 2,\n  \"data\": [\n    {\n      \"title\": \"Example story\",\n      \"url\": \"https://news.ycombinator.com\",\n      \"author\": \"alice\",\n      \"points\": 123,\n      \"created_at\": \"2024-01-01 00:00 UTC\"\n    }\n  ]\n}

GET API Information

Get general information about the API.

Endpoint:
/api/info
Response Example:
{
  "name": "Hacker News API",
  "version": "1.0.0",
  "description": "Fetch front page Hacker News stories",
  "endpoints": {...}
}
