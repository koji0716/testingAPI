API Endpoints

GET Health Check

Check if the API is running and healthy.

Endpoint:
/api/health
Response Example:
{
  "status": "healthy",
  "message": "API is running successfully",
  "timestamp": "2025-09-01T00:00:00Z"
}

GET Get All Users

Retrieve a list of all users.

Endpoint:
/api/users
Response Example:
{
  "status": "success",
  "data": [...],
  "count": 2
}

GET Get User by ID

Retrieve a specific user by their ID.

Endpoint:
/api/users/{id}

Parameters:
  • id (integer): User ID
Response Example:
{
  "status": "success",
  "data": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com"
  }
}

POST Create User

Create a new user.

Endpoint:
/api/users

Request Body:
{
  "name": "John Doe",
  "email": "john@example.com"
}
Response Example:
{
  "status": "success",
  "message": "User created successfully",
  "data": {
    "id": 3,
    "name": "John Doe",
    "email": "john@example.com"
  }
}

PUT Update User

Update an existing user.

Endpoint:
/api/users/{id}

Parameters:
  • id (integer): User ID
Request Body:
{
  "name": "Updated Name",
  "email": "updated@example.com"
}
Response Example:
{
  "status": "success",
  "message": "User updated successfully",
  "data": {
    "id": 1,
    "name": "Updated Name",
    "email": "updated@example.com"
  }
}

DELETE Delete User

Delete a user by their ID.

Endpoint:
/api/users/{id}

Parameters:
  • id (integer): User ID
Response Example:
{
  "status": "success",
  "message": "User deleted successfully",
  "data": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com"
  }
}

GET API Information

Get general information about the API.

Endpoint:
/api/info
Response Example:
{
  "name": "Simple Flask REST API",
  "version": "1.0.0",
  "description": "A simple REST API built with Flask",
  "endpoints": {...}
}
